# 🤖 ai-playground

A personal sandbox for experimenting with AI models, APIs, and integrations. This repository serves as a space to explore ideas, prototype features, and test out different approaches to building AI-powered experiences.

## What's here

This repo is intentionally freeform — experiments may range from quick scripts to more structured prototypes. Some areas of exploration include:

- **LLM integrations** — calling APIs from Anthropic, OpenAI, and others
- **Prompt engineering** — testing prompting strategies and patterns
- **AI-powered tooling** — small utilities and automations driven by language models
- **Multiplatform experiments** — exploring AI features in Kotlin/Compose Multiplatform contexts

## Structure

Projects and experiments are organized into directories as they grow. Each one should contain its own README or inline comments explaining what it does and how to run it.

## Getting started

Most experiments are self-contained. Check the directory of the experiment you're interested in for specific setup instructions. Common requirements:

- A valid API key for the model provider being used (set via environment variable or a local `.env` file — never committed)
- Language runtime appropriate for the experiment (JVM, Node.js, Python, etc.)

## Notes

This is a personal learning space. Code here is exploratory and may not follow production standards. Things may break, change, or disappear as experiments evolve.
